# marginalia

> ダウンロードした PDF / EPUB を「紙の本のように」読むための iPhone リーダー。
> 自分で出版するのではなく、手元の本を読みやすく表示し、メモを書き、見返すためのツール。

## なぜ作るのか

Kindle をはじめとする電子書籍は、デジタルの便利さと引き換えに **紙の本が持っていた身体的な感覚** を失っている。「今どのくらい読んだか」「あの話はどのあたりに書いてあったか」「真ん中あたりをパッと開く」——紙なら自然にできるのに、デジタルでは難しい。

これは認知科学でも裏付けられている。複数のメタ分析が、紙のほうが（特に説明的な文章で）読解に優れることを示しており、その一因として **空間記憶**（情報がページ上・本の中のどこにあったかで思い出す力）が挙げられている。詳細は [docs/concept.md](docs/concept.md)。

## コアコンセプト

> **本を「スクロールする文字列」ではなく、「厚みと位置を持つ物体」として扱う。**

紙の身体感覚——厚みで進捗を体感し、位置で内容を思い出し、指でパッと開く——を iPhone 上で取り戻す。

## 差別化の核

- **小口 + 付箋**: 画面右端に本の小口（厚み）を常時表示。メモ・ハイライトが種類ごとに色分けされた付箋として、本の中の位置に刺さる。小口を掴めばその位置へジャンプできる。
- **触覚フィードバック**: パラパラめくると Taptic Engine が反応。付箋を通過する瞬間、種類ごとに異なる触覚が返る——画面を見ずとも指で「あのへんに重要な付箋がある」が分かる。
- **取り込み時にページ確定**: 取り込み時に好みの文字サイズでレイアウトを確定し、以後は不変。位置が動かないので空間記憶が完全に効く（Kindle の弱点を回避）。
