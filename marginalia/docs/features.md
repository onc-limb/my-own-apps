# 機能一覧

優先度はフェーズで表現する（[roadmap](../roadmap/README.md) 参照）。すべての機能は「本を厚みと位置を持つ物体として扱う」という1本の軸から導かれる。

## コア: 小口 + 付箋（MVP）

- 画面右端に**小口（厚みゲージ）を常時表示**。既読/未読を厚みで体感する。
- 小口を掴んでドラッグでその位置へジャンプ（「真ん中あたりを開く」操作）。
- ハイライト/メモを**付箋**として本の中の位置に表示。
- 付箋は**メモの種類で色分け**（重要 / 疑問 / 引用 / あとで読む など）。
- 付箋・メモは**本文テキストのアンカーに紐付け**（再レイアウトしても位置がズレない）。

## 取り込み・表示（MVP）

- PDF / EPUB の取り込み（ファイル App / 共有シート）。
- **取り込み時に好みの文字サイズでレイアウトを確定**し、以後は不変（位置を安定させ空間記憶を効かせる）。
- PDF は中身で 2 系統に分岐:
  - **テキスト主体** → 取り込み時に再レイアウト & 固定。
  - **画像/複雑レイアウト**（スキャン・図表/数式の多い論文）→ 原本維持 + 余白トリミング + 列フィット。
- EPUB は取り込み時に 1 回リフローして固定ページ化（PDF と同じ「動かないページの集合」に正規化）。
- 進捗（どこまで読んだか）の記録。

## パラパラ読み + 触覚（MVP）

- 小口ジェスチャーの**速度連続体**: ゆっくり = 精密ジャンプ / 速く擦る = スキミング、指を止めれば開く。
- **触覚フィードバック（Core Haptics / Taptic Engine）**:
  - ページ送りの微細な tick（速度に連動）。
  - **付箋通過の触覚を種類ごとにパターンで変える**（画面を見ずに位置が分かる）。
  - 章境界でのスナップ / 始端・終端での壁。
- **紙めくり効果音（オプション）**。触覚・音とも設定でオフ可。

## 学びノート（フェーズ2）

- 付箋（ハイライト + メモ）を抜き出した**一覧ビュー**（= 小口の裏返し）。
- 各メモに**位置（本の◯% / ページ）を併記**。タップで小口のその位置へ飛ぶ。
- 種類（色）でフィルタ。

## 本棚 / ライブラリ（フェーズ2）

- 登録した本を一覧。各本の**背表紙が既読分だけ満ちる**（厚みメタファーの棚スケール拡張）。
- 「何をどこまで読んだか」を棚を眺めて把握。

## 発展（フェーズ3）

- 本全体の**俯瞰マップ**（全ページを帯/グリッドで一望、章を色分け）。
- 学びノートの**全書籍横断検索**。
- **Markdown エクスポート** / 間隔反復による復習。
- **読みやすさモード**（テキストだけリフロー、位置は捨てる切替モード）。
