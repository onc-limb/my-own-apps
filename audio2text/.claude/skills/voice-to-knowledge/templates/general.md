# Source Note Template: General

特定のシーンに紐づかない汎用テンプレート。**情報の取りこぼしを最小化することを優先**し、各セクションに該当するものがあれば残す。

## 抽出時の基本方針

- 「何の音声か」を把握し、`## 要約 (TL;DR)` で2〜5文に凝縮
- 議論があるなら `## 話題の流れ` に主要トピック単位で
- 各種セクション（決定 / タスク / アイデア / 学び / 知識・事実 / 問い / 用語）は該当するものがあれば残す
- 該当しないセクションは省略する（プレースホルダーは残さない）
- 後で読み返した時に文脈が分かるよう、`## 引用` を3〜5個残す

## 出力フォーマット

```markdown
---
title: <内容を表す短いタイトル>
source_audio: <元音声ファイル名>
source_transcript: <transcript .md ファイル名>
recorded_at: <YYYY-MM-DD>
processed_at: <ISO8601>
category: general
preset: <使用したプリセット名>
language: <ja|en|...>
duration: <HH:MM:SS>
participants:
  - <名前 or 役割>
project: <プロジェクト名>
tags:
  - <キーワード>
---

# <Title>

## 要約 (TL;DR)

<必須。2〜5文で全体を要約。>

## 話題の流れ

- **<話題1>**: <要点>
- **<話題2>**: <要点>

## 決定事項

- **<決定>**: <内容>

## タスク

- [ ] <内容>

## アイデア

- **<アイデア>**: <内容>

## 学び・気付き

- **<気付き>**: <内容>

## 知識・事実

- ...

## 未解決の問い

- **<問い>**: <背景>

## 用語・概念

- **<用語>**: <定義>

## キーワード

`<keyword1>` `<keyword2>` ...

## 引用 / 重要発言

- `[HH:MM:SS]` <発言>

## 関連

- [[sources/<YYYY-MM>/<related-file>]]
```

## 注意

- 該当する内容がないセクションは見出しごと省略してよい。`## 要約 (TL;DR)` だけは必須
- 個人情報は `[REDACTED]` でマスク
- 推測が混じる場合は `<!-- 推測: ... -->` で明記
- このテンプレートを使うことが多い場合は、専用テンプレートを別途作成することを検討
