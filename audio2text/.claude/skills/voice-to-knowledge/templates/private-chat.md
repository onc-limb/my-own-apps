# Source Note Template: Private Chat

仕事ではない私的な会話を1ファイルで記録する。**個人的な思い出・関係性のメモ帳** として機能させる。

## 抽出時の基本方針

- 「何があったか」「誰が出てきたか」「何を約束したか」が中心
- 業務的な視点（決定 / アクション）は不要。代わりに `## 学び・気付き` で感情の動き・関係性の機微を拾う
- 約束・予定は `## タスク` に
- 第三者の評価・噂は最小限。書く場合も抽象化
- ジョーク・印象的なフレーズは `## 引用` に残しておくと後で読み返した時に楽しい
- センシティブな情報（健康・人間関係の悩み等）は本人の意向に応じて慎重に扱う

## 出力フォーマット

```markdown
---
title: <会話のテーマや出来事を表す短いタイトル>
source_audio: <元音声ファイル名>
source_transcript: <transcript .md ファイル名>
recorded_at: <YYYY-MM-DD>
processed_at: <ISO8601>
category: private-chat
preset: <使用したプリセット名>
language: <ja|en|...>
duration: <HH:MM:SS>
participants:
  - <名前 or 関係性: 例「妻」「友人A」>
mood: <例: relaxed / nostalgic / heated / fun>
tags:
  - <キーワード>
---

# <Title>

## 要約 (TL;DR)

<必須。2〜3文で「誰と何の話をしたか、何が印象的だったか」>

## 話題の流れ

- **<話題1>**: <要点>
- **<話題2>**: <要点>

## 決定事項

<行く先・日程など軽い決定が出ていれば。>

- **<決定>**: <内容>

## タスク

<約束・予定。日時・相手が分かれば併記。>

- [ ] <内容> — 日時: <?>、相手: <?>

## アイデア

<試したいこと、行きたい場所など。>

- **<アイデア>**: <内容>

## 学び・気付き

<会話を通じての感情の流れ、関係性の発見、自分の認識変化。>

- **<気付き>**: <内容>

## 知識・事実

<生活情報、経験談、誰かのプロフィール情報など。>

- ...

## 未解決の問い

<確認したいこと、後で聞きたいこと。>

- **<問い>**: <背景>

## 用語・概念

<会話中に定義された用語・身内ジャーゴン。>

- **<用語>**: <意味>

## キーワード

`<keyword1>` `<keyword2>` ...

## 引用 / 印象的なフレーズ

<3〜5個。残しておきたい言葉・ジョーク。>

- `[HH:MM:SS]` "<発言>"

## 関連

- [[sources/<YYYY-MM>/<related-file>]]
```

## 注意

- 該当する内容がないセクションは見出しごと省略してよい。`## 要約 (TL;DR)` だけは必須
- 第三者についての評価的内容は最小限に。書くなら抽象化
- センシティブな話題が含まれる場合、`tags` に `sensitive` を追加するか、そもそも記録を控える選択も検討
- 推測が混じる場合は `<!-- 推測: ... -->` で明記
