# Summary Templates

`voice-to-knowledge` スキルが要約・知識化する際に使うテンプレート集。

各テンプレートは Claude への指示書として機能し、frontmatter とセクション構成、抽出すべき情報の重みづけが書かれている。

## テンプレート一覧

| ファイル | 想定シーン | 重視する観点 |
|---|---|---|
| `business-meeting.md` | 仕事上の会議・打ち合わせ | 議事録、決定事項、アクションアイテム、責任者・期日 |
| `business-chat.md` | 仕事上の雑談・1on1・廊下会話 | 業務に活きる発見、関係性、ふわっとした情報 |
| `monologue.md` | 独り言・思考メモ・運転中音声 | 思考の流れ、未解決の問い、自分への TODO |
| `private-chat.md` | 友人・家族との雑談 | 出来事、人物、感情、関係性、約束 |
| `interview.md` | インタビュー・対話形式 | Q&A 構造、引用、フォローアップ質問 |
| `lecture.md` | 講演・プレゼン・学習素材 | 章立て、定義、例、要復習事項 |
| `general.md` | 分類できないもの・汎用 | 標準的な要約 |

## 選び方フローチャート

```
仕事に関係する？
├─ Yes
│  ├─ 形式的な打ち合わせ (議題・参加者・決定がある)  → business-meeting
│  ├─ 雑談的・1on1・カジュアル                       → business-chat
│  ├─ Q&A 形式                                       → interview
│  └─ 一方的な発信 (講演・プレゼン・教材)              → lecture
└─ No
   ├─ 自分一人の思考メモ                              → monologue
   ├─ 親しい人との会話                                → private-chat
   └─ 上記に当てはまらない                            → general
```

## カスタマイズ

- テンプレートを追加: 新しい `<name>.md` をこのディレクトリに置き、`presets.json` と `SKILL.md` にエントリを追加
- 既存テンプレートを編集: そのファイルを直接修正（frontmatter / セクション / 抽出ルール）
- 完全に自分専用にする: テンプレートをコピーして `my-<name>.md` を作り、`presets.json` で参照
