# TimeBranch

階層化したプロジェクトごとの作業時間を、iPhone上で記録・集計するSwiftUIアプリです。

## 開発環境

- Xcode 16以降
- iOS 17以降
- Swift 6 / SwiftUI / SwiftData
- XcodeGen

## 起動

```sh
xcodegen generate
open TimeBranch.xcodeproj
```

データは端末内のSwiftDataストアへ保存されます。「レポート」画面の共有ボタンから、選択期間の集計と記録をJSONとして共有できます。

UIは日本語が既定です。「設定」タブの「表示言語」から英語へ切り替えられ、選択内容は次回起動後も維持されます。

## 計測ルール

- 同時に動作する直接の作業記録は1件だけです。
- 別プロジェクトを開始すると、現在の記録を同時刻で終了して新しい記録を始めます。
- 子・孫の記録は集計時にすべての祖先へ加算されます。
- 同じ親を持つ子同士を切り替えた場合、親の時間には隙間が生じません。
